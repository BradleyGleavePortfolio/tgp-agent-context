# handoffs

Canonical handoffs for TGP agents. See the parent `README.md` for the full layout and the stranded-doc rescue backlog.

Files in this directory are referenced from product repos via the path `tgp-agent-context/handoffs/<filename>`. To survive cross-repo CI doc-reference checks, the bare filename should also appear on each product repo's `.agent-doc-allowlist` until cross-repo path resolution is wired into the verifier.
